# Análisis Exploratorio de Datos - ComercioYA

## Contexto
Este proyecto consiste en un análisis exploratorio de datos (EDA) aplicado a un dataset simulado de clientes de una tienda online llamada ComercioYA.  
El objetivo es comprender el comportamiento de los clientes, identificar patrones y relaciones entre variables, y extraer insights útiles para la toma de decisiones comerciales.

## Dataset
- 300 registros de clientes ficticios
- Variables: visitas al sitio, número de compras, monto gastado, devoluciones y calificación del servicio

## Metodología
1. Generación y exploración inicial del dataset
2. Estadística descriptiva
3. Análisis de correlación
4. Modelado mediante regresión lineal
5. Visualización avanzada con Seaborn y Matplotlib

## Resultados principales
- Las variables presentan distribuciones variadas, con un valor atípico de monto gastado cercano a 160.000
- No se encontraron correlaciones fuertes entre variables
- La regresión lineal mostró que el número de compras no explica significativamente el monto gastado

## Conclusiones y recomendaciones
- El comportamiento de gasto no se correlaciona con el número de compras
- Se recomienda profundizar en análisis de segmentación de clientes por calificación y monto gastado
- Visualizaciones claras facilitan la interpretación y comunicación de insights
